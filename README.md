## About the Project
This project is a responsive (mobile-friendly) e-commerce website developed using HTML, CSS, and JavaScript for the promotion and sale of handmade natural candles. Users can browse products, search for specific items, add products to their cart, and contact the store through a contact form.

##  Technologies and Tools Used
* **HTML5:** Semantic tag structure was used.
* **CSS3:** Responsive design was achieved using Flexbox and Media Queries.
* **JavaScript:** Used for cart operations, product search/filtering, modal (popup) functionality, and menu control.
* **Font Awesome:** Used for icons (cart, search, and social media icons).
* **Google Fonts:** “Playfair Display” and “Roboto” fonts were used for typography.

## Key Features
* **Responsive Menu:** A side navigation menu on mobile and a dynamic header menu on desktop.
* **Product Search:** Real-time product filtering using keywords on the `products.html` page.
* **Shopping Cart System:** Ability to add products to the cart, view the total amount, and display selected items.
* **Modal Structure:** Product details and cart contents are displayed in a popup window without refreshing the page.
